## homebrew的使用

### 安装
```sh
ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"
```

### 搜索指定软件

```sh
brew search xxx
```

### 查询指定软件信息

```sh
brew info xxx
```

### 更新homebrew自己

```sh
brew update
```

### 检查软件所有安装的软件是否有新版本

```sh
brew outdated
```

### 升级所有可以升级的软件

```sh
brew upgrade
```

### 升级指定软件

```sh
brew upgrade xxx
```

### 清理不需要的版本及安装包缓存

```sh
brew cleanup
```
